Important Security Notice

If your server has the /script command enabled, your server is or was vulnerable to a file manipulation exploit. This exploit affects Minecraft version 1.21.4-226. An attacker could have used this vulnerability to sneak malicious code into your scripts. To fix this, carefully review all your scripts and remove any usage of the /script command .

To check your server version, use the command /ver This is pached on 1.21.4-227.
